# Chinese Speech Trainer — MVP

## Goal
A lightweight browser app (mobile + desktop) to improve spoken Chinese:
- tones & pinyin accuracy
- natural pronunciation through shadowing
- basic character recognition tied to spoken phrases

## MVP Scope (must-have)
### Drills (only 3)
1) Tone drill (minimal pairs)
2) Shadowing drill (listen → repeat)
3) Hanzi recognition drill (SRS flashcards)

### Session rule
- A session is 10 minutes.
- The UI should always offer: “Start 10-minute session”.

### Feedback rule (critical)
- Show **max 3 bullet points** of feedback per attempt.
- Feedback must be **actionable** (what to change + how).
- No long explanations.

## Out of scope (for MVP)
- Grammar lessons
- Writing/handwriting
- Large course trees
- Social features
- Long recordings (> 10 seconds per attempt)

## Data rules
- Local-first: user progress stored in the browser.
- Content starts small (≈ 30 items) and can grow later.
